# es6-practice
This is a simple es6 practice environment
传送门 [es6-practice webpack版](https://github.com/hjzheng/es6-practice-webpack)

### Usage

- `npm install`
- `npm run serve`
- Write es6 code in `src/*.js` files, if have new js file, remember import it in `index.js` file.
